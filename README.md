# Library Catalog
A simple web app that visualizes the openlibrary API data. The user is able to search a book title or keyword via the input search bar and upon search, will see ten query results render to the page. 

## Technologies 

- HTML5
- CSS3
- JavaScript
- jQuery
- Open Library API
